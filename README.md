<h1 align="center">HttpCache 1.2.0</h1>

<p align="center">Cached HTTP requests for GameMaker 2024.14 by <a href="https://www.jujuadams.com/" target="_blank">Juju Adams</a></p>

<p align="center"><a href="https://github.com/JujuAdams/httpcache/releases/">Download the .yymps</a></p>

&nbsp;

Adds caching for HTTP requests to a GameMaker project. On Windows, MacOS, Linux, iOS, and Android, HttpCache will save the result of successful HTTP requests to disk. On other platforms (or if you disable disc caching) then HTTP requests will be cached in RAM. All HTTP request functions use function callbacks instead of GameMaker's clumsy async event system.
